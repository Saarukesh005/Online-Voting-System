# Online Voting System (Java Swing UI)

## 📌 Description

This is a simple Online Voting System developed using Core Java and Swing for UI. The system allows users to vote securely and prevents duplicate voting. Results are hidden until the admin ends the voting.

## 🚀 Features

* GUI-based voting system (Java Swing)
* Secure voting (results hidden during voting)
* Admin control to end voting
* Prevents duplicate voting
* Displays winner after voting ends

## 🛠 Technologies Used

* Java
* Swing (JFrame, JButton, JOptionPane)
* OOP Concepts

## ▶️ How to Run

1. Compile:
   javac src/VotingUI.java

2. Run:
   java -cp src VotingUI
